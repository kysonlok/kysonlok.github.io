Gitee搭建个人图床

写在前面

利用 PicGo 工具把本地图片上传到 Gitee，来实现个人图床。选择 Gitee 的原因是因为在国内访问 Gitee 较快，访问 GitHub 是蜗牛的速度。PicGo 默认不支持 Gitee 作为图床，要安装 gitee-uploader 插件。
